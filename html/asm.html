First: What is assembly?

Assembly is a really low level language, it uses a set of instructions to tell the CPU what to directly, the programmer must memorize these instructions to manipulate the memory and the CPU registers in specific ways to get his desired output.

Second: Assembly's syntax is affected by four things

  1. Your CPU architecture.
  2. Your OS.
  3. Your assembler.
  4. Wether you're using 64, 32 or 16 bit registers.
